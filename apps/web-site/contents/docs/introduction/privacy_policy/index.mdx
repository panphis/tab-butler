# [用户隐私声明协议](#用户隐私声明协议)

## 感谢您使用 Tab Butler 插件

我们非常重视您的隐私和数据安全。以下是我们对用户隐私的承诺和声明：

## 1. 数据收集

Tab Butler 插件在使用过程中可能会收集以下数据：

- 浏览历史记录（history）
- 书签信息（bookmarks）
- 剪贴板内容（clipboardRead, clipboardWrite）
- 活动标签页信息（activeTab, tabs, tabCapture, tabGroups）
- 顶部网站信息（topSites）
- 网站图标（favicon）
- 存储数据（storage）
- 上下文菜单信息（contextMenus）
- 内容设置（contentSettings）
- 本地消息传递（nativeMessaging）

## 2. 数据使用

我们收集的数据仅用于以下目的：

- 提供和改进插件功能
- 个性化用户体验
- 处理用户请求和反馈

## 3. 数据存储

所有数据都存放在用户本地电脑上且不会上传到互联网。我们不会与任何第三方共享您的个人数据，除非在以下情况下：

- 获得您的明确同意
- 法律要求

## 4. 数据安全

我们采取合理的技术和组织措施来保护您的数据免受未经授权的访问、使用或披露。

## 5. 用户权利

您有权访问、更正或删除我们收集的您的个人数据。您可以通过插件设置页面或联系我们来行使这些权利。

## 6. 变更通知

我们可能会不时更新本隐私声明。任何更改将在插件更新时通知您。

# [Privacy Policy](#privacy-policy)

## Thank you for using the Tab Butler extension

We highly value your privacy and data security. The following is our commitment and statement regarding user privacy:

## 1. Data Collection

The Tab Butler extension may collect the following data during use:

- Browsing history (history)
- Bookmarks (bookmarks)
- Clipboard content (clipboardRead, clipboardWrite)
- Active tab information (activeTab, tabs, tabCapture, tabGroups)
- Top sites information (topSites)
- Website icons (favicon)
- Storage data (storage)
- Context menu information (contextMenus)
- Content settings (contentSettings)
- Local messaging (nativeMessaging)

## 2. Data Usage

The data we collect is used solely for the following purposes:

- Providing and improving extension functionality
- Personalizing user experience
- Handling user requests and feedback

## 3. Data Storage

All data is stored on the user's local computer and will not be uploaded to the internet. We do not share your personal data with any third parties, except in the following cases:

- With your explicit consent
- As required by law

## 4. Data Security

We take reasonable technical and organizational measures to protect your data from unauthorized access, use, or disclosure.

## 5. User Rights

You have the right to access, correct, or delete your personal data that we collect. You can exercise these rights through the extension settings page or by contacting us.

## 6. Change Notification

We may update this privacy statement from time to time. Any changes will be notified to you when the extension is updated.
