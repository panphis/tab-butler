## [已知问题](#已知问题)

### [安装包体积过大](#安装包体积过大)

安装包中有一个默认高质量背景图 （14M），安装过慢可以尝试下载手动打包安装。

### [网页图标显示不出来](#网页图标显示不出来)

本插件采用`Manifest v3`方式，通过`chrome-extension://EXTENSION_ID/_favicon/?pageUrl=EXAMPLE_URL&size=FAV_SIZE`方式获取网站缓存图片，如果显示不出来，则可能遇到一下场景。

1. 该网站图标还未被浏览器缓存。
2. 某些网站在`Manifest v3`版本中无法获取图标 [IssueTracker](https://issuetracker.google.com/issues/334120886)

### [插件申请权限过多](#插件申请权限过多)

本插件采用完全离线方式获取谷歌浏览器缓存数据，所申请权限均为书签，历史搜索记录等功能正常使用。所有功能均可离线使用，**绝无任何数据上传服务器等有造成用户隐私数据泄露等操作**。
