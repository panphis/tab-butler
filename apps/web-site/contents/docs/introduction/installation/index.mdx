使用如下命令依赖是否被安装成功

```bash
node -v
# version for nodejs

pnpm -v
# version for pnpm
```

```bash
git --version
```

#### 安装

<Step>
<StepItem title="下载项目">
首先, 需要使用`git` clone 代码到本地:

```bash
git clone https://github.com/panphis/tab-butler
```

或者通过点击下载来下载代码。

</StepItem>
<StepItem title="安装依赖">

进入到项目目录，执行安装依赖命令

```bash
pnpm install
```
</StepItem>

<StepItem title="运行构建依赖包命令">

在真正运行项目前需要将多仓库内的依赖先构建出来。并再次执行安装命令，将 `package` 中构建好的依赖 安装到 `app` 项目中。

```bash
pnpm run ready

pnpm i
```
</StepItem>

<StepItem title="打包">

在项目根目录下运行构建命令。生成一个 `dist` 文件夹

```bash
# 打包测试环境
pnpm run dev 

# 打包生产环境
pnpm run build
```

</StepItem>


<StepItem title="安装">

1. 打开`[chrome://extensions/](chrome://extensions/ 'chrome://extensions/')` 进入浏览器插件管理页面
2. 点击 `开发者模式` 进入到开发模式。
3. 点击 `加载未压缩的安装宝` 弹出加载窗口。
4. 选择上一步生成的文件夹。
</StepItem>

</Step>
