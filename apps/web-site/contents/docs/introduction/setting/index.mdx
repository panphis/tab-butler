## 设置

### [首页背景，以及固定常用网址](#首页背景，以及固定常用网址)

![首页](/images/home.png "首页")

- 点击搜索下拉，可实现搜索引擎切换。
- 输入任意内容，可优先从书签/历史记录中找到网站地址并显示出来。
- 右键点击网站图标，可复制，打开，编辑，移除。
- 鼠标覆盖到网站图标上，点击省略号，可实现，固定/解除固定，移除。
- 点击 + 可以添加网站到首页

### [书签目录搜索](#书签目录搜索)

![书签](/images/bookmark.png "书签")

- 点击左上角浏览器拓展设置，选中 ![logo](/images/icon-16.png "logo") 固定插件到浏览器工具栏。再次点击可以弹出**浏览器本地缓存**的书签记录。
- 点击书签可以快速打开一个新窗口，访问书签网站。
- 右键书签可弹出书签信息进行修改或者删除操作。
- 可通过拖拽实现书签所在位置的修改。

### [任意网站保存到书签](#任意网站保存到书签)

![保存书签](/images/save_bookmark.png "保存书签")

- 在任意网站右键，点击 `保存到书签` 可快速添加当前网站到**浏览器本地缓存**的书签记录。

### [主题色背景图/视频设置](#主题色背景图/视频设置)

![外观设置](/images/appearance.png "外观设置")

- 通过设置对应选项选择显示语言，主题，主题色等外观
- 通过上传图片/视频，改变浏览器新tab页壁纸。在壁纸列表中可对壁纸信息进行修改

### [搜索引擎，额外搜索参数处理](#搜索引擎，额外搜索参数处理)

![搜索参数](/images/search_engine.png "搜索参数")

- 可以通过复制网站 url 添加搜索引擎
- 可以通过配置搜索引擎，永久添加额外的搜索参数。

#### [关于添加搜索引擎](#关于添加搜索引擎)

以 github 为例

1. 打开GitHub 搜索 chrome 得到页面 url 地址
   `https://github.com/search?q=chrome&type=repositories`

2. 删除无用参数并去掉上一步搜索关键次得到我们所需要的搜索地址
   `https://github.com/search?q=`

常用搜索引擎

| 名称   | 搜索地址                         |
| :----- | :------------------------------- |
| google | https://www.google.com/search?q= |
| Bing   | https://bing.com/search?q=       |
| 必应   | https://cn.bing.com/search?q=    |
| 百度   | https://www.baidu.com/s?wd=less  |
| GitHub | https://github.com/search?q=     |
| npm    | https://www.npmjs.com/search?q=  |

#### [关于搜索参数](#关于搜索参数)

关于搜索参数，需要根据各个搜索引擎使用文档。针对设置。并不是所有搜索引擎设置都可以生效。

1. google:[谷歌高级搜索](https://www.google.com/advanced_search?q=)
2. Bing/必应:[必应高级搜索](https://support.microsoft.com/zh-cn/topic/%E9%AB%98%E7%BA%A7%E6%90%9C%E7%B4%A2%E9%80%89%E9%A1%B9-b92e25f1-0085-4271-bdf9-14aaea720930)
3. 百度: 百度首页 > 设置 > 高级搜索

以排除搜索结果包含 csdn CSDN 为例

google ` -csdn -CSDN` (注意前端有个空格)

Bing/必应 ` -site:csdn.net` (注意前端有个空格)

百度 ` -(csdn | CSDN)` (注意前端有个空格)
